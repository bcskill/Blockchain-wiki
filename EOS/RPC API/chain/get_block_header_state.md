# chain/get_block_header_state

`chain/get_block_header_state`调用返回指定区块的头信息

注意，nodeos时需要启用chain_api_plugin插件,且只能查询head_block_num与lase_irreversible_block_num之前的id

### 调用参数
JSON对象，用来指定要获取块头信息的区块，成员如下：

- block_num_or_id：区块序号或ID，字符串

### 返回值
`get_block_header_state`调用返回一个描述区块头信息的JSON对象

### 示例代码
调用请求：
```shell
~$ curl -X POST --url https://api.eoslaomao.com/v1/chain/get_block_header_state -d '{
  "block_num_or_id": "4"
}'
```
